# ML / Data Science

Эксперименты, ноутбуки и пайплайны — по подпапке на идею:

- `i6-flood-risk/` — Sentinel-1/2, Copernicus DEM, скоринг паводкового риска
- `i9-fire-risk/` — NASA FIRMS + ERA5, скоринг пожарного риска

Внутри каждой: `notebooks/` для экспериментов, `pipeline/` для воспроизводимого кода. Результаты (скоры, GeoJSON) выгружаем в PostgreSQL или в `data/processed/` для загрузки бэкендом. Каждый датасет описываем в `docs/data/` — это требование ТЗ (раздел 9).
