# Frontend — Vue 3

Единый веб-клиент. Инициализация:

```bash
npm create vue@latest .
npm install
npm run dev
```

Идеи оформляем как модули: `src/modules/i6-flood-risk/`, `src/modules/i9-fire-risk/`, `src/modules/poaching/` — каждый со своими страницами, компонентами и API-клиентом. Общее (layout, карта, UI-kit) — в `src/shared/`.
