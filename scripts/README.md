# Скрипты

Запуск стека — через `make up` в корне репозитория (см. Makefile), отдельного скрипта нет.

Сюда складывать вспомогательные скрипты: скачивание датасетов (Sentinel, FIRMS, ERA5), загрузка метрик в API, генерация отчётов. Один скрипт — одна задача, имя вида `download_firms.py` / `load_scores.py`.
