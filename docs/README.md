# Документы

Единственное место для документов проекта. Тематические подпапки:

- [org/](org/) — договор участия, ТЗ отборочного этапа, дедлайны, организационные заметки
- [ideas/](ideas/) — проработка идей: по одной подпапке на идею (`i6-flood-risk`, `i9-fire-risk`, `poaching`)
- [architecture/](architecture/) — архитектура решения, ADR, схема БД, контракты API
- [data/](data/) — описание датасетов: источник, структура, ограничения, обработка, признаки (требование ТЗ, раздел 9)
- [presentation/](presentation/) — презентация 7–10 слайдов, сценарий демо-видео

Не кладите документы в корень репозитория или в папки кода.
