# И-9. Степные/лесные пожары: риск-карта и раннее оповещение

Скоринг районов по пожарному риску (сухостой, ветер, история очагов) + очередь патрулей.

- **Боль КЗ:** ежегодные степные пожары (Абайская обл., 2023).
- **Данные:** NASA FIRMS (очаги, открытый API) + ERA5-метео — настоящие открытые данные без разрешений.
- **Реюз:** ~60%.
- **Скоуп на 8 дней:** средний. Тёмная лошадка с реальными данными.
