# И-6. Паводковый риск-скоринг ⭐ фаворит

Риск-карта населённых пунктов к весеннему паводку + очередь превентивных мер (ДЧС/акимат).

- **Боль КЗ:** паводки-2024 — «крупнейшее бедствие за 80 лет»: >118 тыс. эвакуированных, ущерб ~₸400 млрд.
- **Мировой аналог:** Google Flood Hub — 150+ стран, прогноз до 7 дней.
- **Данные:** Sentinel-1/2 + Copernicus DEM (открытые); разметка зон затопления 2024 — своя.
- **Реюз:** ~50–60% (+опыт NDVI).
- **Скоуп на 8 дней:** сузить до одного региона (СКО или Актюбинская обл.).

Подробно: [value-and-ml.md](value-and-ml.md) — кому продавать, боль, AI/ML-подход; [plan.md](plan.md) — план работ и прогноз баллов по критериям ТЗ.
